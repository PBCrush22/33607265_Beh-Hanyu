Instructions

Welcome to Tetris! Here's how to play:

  1. Use the W key to rotate the Tetromino.
  2. Use the A key to move the Tetromino to the left.
  3. Use the S key to move the Tetromino down faster.
  4. Use the D key to move the Tetromino to the right.
  5. Clear rows to earn points and level up.
  6. Obstacles may appear as the game gets harder!
  7. Click the Restart button to restart the game anytime.
  8. Try to achieve the highest score and level!
